**LeetCode Meme Extension**

A small Chrome extension that adds a fun reaction when you run LeetCode test cases.

What it does
All tests pass → shows a success reaction
All tests fail → shows a confused reaction
Mixed results → no reaction

How it works

It watches the test result icons on the page and briefly shows a reaction based on whether they turn green or red.

To use it - 

Download the folder
Go to chrome://extensions

Enable Developer mode

Click Load unpacked and select the folder

Enjoy LeetCoding with interactive memes!!
